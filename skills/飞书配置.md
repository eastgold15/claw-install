# 🔧 飞书配置详细说明

> OpenClaw 飞书集成的详细配置步骤和问题排查

## 📋 配置流程详解

看这配置流程，你这不是在弄 NapCat，而是正在配置 **OpenClaw** 或类似的 **AI Agent 框架**，而且是在配置**飞书（Feishu）**这个渠道的账号。

这里把每一行的意思和作用都给你拆解清楚：

### ⚙️ Configure DM access policies now? (default: pairing)

- **翻译**: 现在配置私聊（Direct Message）访问策略吗？
- **意思**: 问你是否要设置谁能私聊机器人。
- **你的选择（No）**: 跳过详细设置，使用默认策略（通常意味着允许任何人私聊）。
- **作用**: 安全控制。如果选 Yes（配对模式），陌生人私聊机器人时，机器人可能不会回复，或者需要先输入密码/指令才能开始对话，防止被骚扰。

### ✏️ Add display names for these accounts? (optional)

- **翻译**: 为这些账号添加显示名称吗？（可选）
- **意思**: 问你是否要给这个机器人起个好听的名字，方便在后台管理时识别。
- **你的选择（Yes）**: 要起名。

### 📝 feishu account name (default)

- **翻译**: 飞书账号的名称
- **意思**: 给这个飞书机器人起个什么名字？
- **你的输入（claw）**: 你把它命名为 `claw`。以后在日志或后台看到 `claw`，你就知道是这个飞书账号。

### 🔗 Bind configured channel accounts to agents now?

- **翻译**: 现在将配置好的渠道账号绑定到智能体（Agent）上吗？
- **意思**: OpenClaw 的核心是 Agent（大脑）。这一步是问你：刚配好的这个飞书账号，要连接到哪个大脑去干活？
- **你的选择（Yes）**: 要绑定。

### 🛣️ Route feishu account "default" to agent

- **翻译**: 将飞书账号 "default" 的消息路由到哪个智能体？
- **意思**: 既然要绑定，那就指定一下具体绑定到哪一个 Agent。
- **你的选择（main）**: 你选择了名为 `main` 的 Agent。这意味着，所有发给这个飞书机器人（claw）的消息，都会交给 `main` 这个大脑去处理，处理完再由飞书发回去。

### 📥 Routing bindings

- **翻译**: 路由绑定
- **显示内容（Added: feishu accountId=default）**:
    - **意思**: 系统提示你，绑定成功了。
    - **解释**: ID 为 `default` 的飞书账号，现在已经成功连接到了你的系统里，并且被分配去工作了。

## 📌 完整配置总结

你刚才完成了一次 **"飞书机器人接入"** 的配置：

1.  你忽略了一些高级安全设置（私聊策略），直接用默认的。
2.  你给这个机器人起了个名叫 **`claw`**。
3.  你告诉系统：**"飞书上收到的消息，全部交给 `main` 号大脑去处理。"**

现在，你的 AI Agent 已经可以在飞书上收发消息了。

## 🚀 完整配置步骤

### 1. 准备工作

#### 飞书开放平台配置
1. **访问开放平台**: https://open.feishu.cn/app
2. **创建应用**:
   - 点击 "创建应用"
   - 选择 "机器人" 类型
   - 填写应用信息
3. **配置权限**:
   - 在 "权限管理" 中添加必要权限
   - 参考 [[飞书.md]] 中的权限配置

#### 获取应用凭证
1. **App ID**: 在应用概览页面获取
2. **App Secret**: 在 "凭证与基础信息" 中获取
3. **记录保存**: 安全保存这些凭证信息

### 2. OpenClaw 配置

#### 添加飞书渠道
```bash
# 添加飞书渠道
openclaw channels add feishu
```

#### 配置应用信息
```bash
# 设置 App ID
openclaw config set channels.feishu.accounts.default.appId "cli_a927a83881b85bc2"

# 设置 App Secret
openclaw config set channels.feishu.accounts.default.appSecret "cli_a927a83881b85bc2"

# 启用账号
openclaw config set channels.feishu.accounts.default.enabled true
```

#### 配置消息策略
```bash
# 设置私聊策略
openclaw config set channels.feishu.accounts.default.dmPolicy "open"

# 设置群聊策略
openclaw config set channels.feishu.accounts.default.groupPolicy "open"

# 设置允许的用户
openclaw config set channels.feishu.accounts.default.allowFrom ["*"]

# 设置允许的群组
openclaw config set channels.feishu.accounts.default.groupAllowFrom ["*"]
```

#### 绑定智能体
```bash
# 绑定到主智能体
openclaw bind feishu default main
```

### 3. 测试验证

#### 启动服务
```bash
# 启动 OpenClaw Gateway
openclaw gateway start

# 启动 Dashboard
openclaw dashboard
```

#### 测试机器人
1. **在飞书中搜索机器人**: 搜索 "claw"
2. **添加机器人**: 点击添加到联系人
3. **发送测试消息**: 发送 "你好" 或其他测试消息
4. **检查回复**: 确认机器人正常回复

#### 查看日志
```bash
# 查看飞书相关日志
openclaw logs --channel feishu

# 实时查看日志
openclaw logs --follow --channel feishu
```

## 🔧 高级配置

### 1. 多账号配置

如果你需要配置多个飞书账号：

```bash
# 添加第二个账号
openclaw config set channels.feishu.accounts.work.appId "your_work_app_id"
openclaw config set channels.feishu.accounts.work.appSecret "your_work_app_secret"

# 绑定到不同的智能体
openclaw bind feishu work work-agent
```

### 2. 自定义回复策略

```json
{
  "channels": {
    "feishu": {
      "accounts": {
        "default": {
          "replyStrategy": {
            "private": {
              "mode": "auto",
              "delay": 1000,
              "typingIndicator": true
            },
            "group": {
              "mode": "mention",
              "delay": 2000,
              "typingIndicator": false
            }
          }
        }
      }
    }
  }
}
```

### 3. 网络配置

如果需要代理访问：

```bash
# 设置代理
openclaw config set channels.feishu.proxy.enabled true
openclaw config set channels.feishu.proxy.host "proxy.example.com"
openclaw config set channels.feishu.proxy.port 8080
```

## 🚨 故障排除

### 1. 连接问题

#### 问题：WebSocket 连接失败
**错误信息**: `Connection failed` 或 `WebSocket error`

**解决方案**:
1. 检查网络连接
2. 验证 App ID 和 Secret
3. 检查防火墙设置
4. 查看详细日志

```bash
# 检查网络连接
ping api.feishu.cn

# 查看详细错误日志
openclaw logs --level error --channel feishu
```

### 2. 权限问题

#### 问题：权限不足
**错误信息**: `Permission denied` 或 `Insufficient permissions`

**解决方案**:
1. 检查飞书开放平台权限配置
2. 确保应用已发布
3. 重新授权应用
4. 联系飞书技术支持

### 3. 消息处理问题

#### 问题：消息发送失败
**错误信息**: `Message send failed` 或 `No response`

**解决方案**:
1. 检查智能体绑定状态
2. 验证消息策略配置
3. 检查 OpenClaw 服务状态
4. 查看智能体日志

```bash
# 检查智能体状态
openclaw agent status

# 检查服务状态
openclaw gateway status
```

### 4. 配置问题

#### 问题：配置文件错误
**错误信息**: `Config validation failed` 或 `Invalid configuration`

**解决方案**:
1. 验证配置文件语法
2. 检查必需的配置项
3. 使用配置验证工具

```bash
# 验证配置
openclaw config validate

# 查看当前配置
openclaw config show
```
